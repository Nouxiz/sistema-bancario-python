# 🏦 Sistema Bancário em Python (POO)

Este é um projeto de simulação de sistema bancário desenvolvido para praticar conceitos de **Programação Orientada a Objetos (POO)** em Python.

## 🚀 Funcionalidades
* **Depósito:** Permite adicionar valores ao saldo da conta.
* **Saque:** Realiza retiradas respeitando o saldo disponível e limites.
* **Extrato:** Exibe todas as movimentações realizadas na conta.
* **Histórico:** Registro detalhado de transações.

## 🛠️ Tecnologias Utilizadas
* **Python**
* Conceitos de **Classes, Métodos, Encapsulamento e Herança**.
